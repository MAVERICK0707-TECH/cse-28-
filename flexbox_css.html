In the above example, we have created a flex container using the display property set to flex. The flex-direction property is set to row, which arranges the flex items (Item 1 and Item 2) in a horizontal row. The justify-content property is set to space-around, which distributes the items evenly with space around them. The align-items property is set to center, which vertically centers the items within the container.
